# Brain Tumor Classification using CNN (ResNet-18 + MC Dropout)

## Problem Description

Brain tumors are among the most life-threatening medical conditions, and early accurate classification is critical for treatment decisions. This project tackles **multi-class brain tumor classification** from MRI images using deep learning.

The model distinguishes between **4 classes**:

| Class | Description |
|-------|-------------|
| `glioma` | Malignant brain tumor originating in glial cells |
| `meningioma` | Tumor arising from the meninges (brain membranes) |
| `notumor` | Healthy brain MRI with no tumor |
| `pituitary` | Tumor located in the pituitary gland |

---

## Dataset

**Brain Tumor MRI Dataset** — Kaggle

**Dataset Link:** [https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset](https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset)

The dataset contains MRI scan images organized into `Training/` and `Testing/` folders, each with 4 subfolders (one per class).

**Dataset Split used in this project:**

| Split | Samples |
|-------|---------|
| Train | 4,480 |
| Validation | 1,120 |
| Test | 1,600 |
| Critic Dataset (MC Dropout pipeline) | 7,200 |

> **Critic Dataset CSV:** The `critic_dataset.csv` (generated by the MC Dropout pipeline in Section 7 of the notebook) is saved to Google Drive at `/content/drive/MyDrive/critic_dataset.csv`. A full schema and all numeric results are in [`results.csv`](./results.csv).

## Model Architecture

- **Base model:** ResNet-18 pretrained on ImageNet (**backbone frozen**)
- **Custom classification head:**
  ```
  Dropout(0.5) → Linear(512 → 256) → ReLU → Dropout(0.5) → Linear(256 → 4)
  ```

| Parameter | Value |
|-----------|-------|
| Total parameters | 11,308,868 |
| Trainable parameters | 132,356 (only 1.2%) |
| Device | CUDA (GPU) |

**Techniques used:** Transfer Learning · Dropout Regularization · Data Augmentation (Flip + Rotation ±15°) · ReduceLROnPlateau scheduler · MC Dropout Uncertainty Estimation
